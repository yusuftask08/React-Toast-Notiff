## Run Locally

Clone the project

```bash
  git clone https://github.com/yusuftask08/React-Toast-Notiff.git
```

Go to the project directory

```bash
  cd react-toast-notiff
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
